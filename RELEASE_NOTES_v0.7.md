## Release version 0.7

A modern macOS GUI application for OCRMyPDF with drag-and-drop support and batch processing.

### Features
- Drag & Drop Support
- Batch Processing with progress tracking
- Real-time Output
- Multi-language OCR support (8 languages)
- Configurable OCR options
- Siri & Shortcuts Integration
- Modern macOS design

### Requirements
- macOS 13.5 (Ventura) or later
- Homebrew with ocrmypdf and tesseract-lang installed

### Installation
1. Download the `ocrmypdf-gui-v0.7.zip` file
2. Extract the `ocrmypdf-gui.app` from the zip
3. Drag it to your Applications folder
4. Make sure you have `ocrmypdf` and `tesseract-lang` installed via Homebrew

See README.md for full documentation.
